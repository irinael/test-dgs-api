## Description

MS allowing to check the validity of bank operations accoring to checkpoints

Applicaton runs locally on http://localhost:3000

Documentation available here : http://localhost:3000/api (after npm run start)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev


## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
